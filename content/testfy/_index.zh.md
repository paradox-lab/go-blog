# 自动化测试

**生成测试报告:** [go-test-report](https://pkg.go.dev/github.com/meyoustu/go-test-report#section-readme)

## 安装

```text
go install github.com/vakenbolt/go-test-report@latest
```

## go test

### 递归测试所有目录

```text
go test ./...
```
