包管理¶

wiki

https://github.com/golang/go/wiki/Modules

演示用到的工程目录结构

go.mod
go.sum
repo\main.go
     myPackage\myCode.go
     util.go 'package repo
     util_test.go

注解

每个包最多只能存在一个main.go文件

初始化¶

go mod init github.com/my/repo

如果不放远程仓库，可以省略仓库地址

go mod init repo

引用自定义包¶

import "repo/repo/myPackage"
