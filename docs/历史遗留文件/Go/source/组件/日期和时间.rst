**********************
日期和时间
**********************

.. highlight:: go

格式化当前时间
======================

::

    time.Now().String() // 默认值"2006-01-02 15:04:05"
    time.Now().Format("2006-01-02 15:04:05")

解析字符串时间
======================

::

    d, _ := time.ParseDuration("1h30m")

日期和时间加减
=======================

::

    time.Now().AddDate(1, 1, 1))
    time.Now().Add(d))
